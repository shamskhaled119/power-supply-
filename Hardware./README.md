## Hardware Part 🔧⚡

The hardware side of this Adjustable Power Supply is responsible for converting, regulating, displaying, and backing up the output power.

The circuit provides fixed 5V and 12V outputs, along with an adjustable output voltage that can be changed depending on the circuit needs.

It also includes a display module to show the adjustable voltage value in real time, and a battery backup system that keeps the power supply working if the main power is disconnected.

In simple words:  
the circuit regulates the voltage, the display shows the output, and the battery keeps it alive when power cuts. 🔥

## Hardware Components Used

- Voltage Regulators  
  Used to generate stable fixed output voltages such as 5V and 12V.

- Adjustable Voltage Regulator  
  Used to provide a variable DC output voltage depending on the required value.

- Display Module / Voltmeter  
  Used to show the adjustable output voltage in real time.

- Battery Backup Circuit  
  Used to keep the power supply operating when the main power source is disconnected.

- Relay  
  Used to switch or control the power path between the main supply and backup source.

- Capacitors  
  Used for filtering and stabilizing the voltage outputs.

- Resistors  
  Used for voltage adjustment, biasing, and control.

- Terminal Connectors  
  Used to connect the input source, output lines, and external circuits.

## Hardware Role

- Converts the input power into usable DC outputs
- Provides fixed 5V output
- Provides fixed 12V output
- Provides adjustable output voltage
- Displays the adjustable voltage value
- Supports battery backup operation
- Helps power electronic circuits safely and reliably
